# Ambient Sound Files

This folder contains ambient sound loops for the Palante app.

## Required Files:

Download and add these MP3 files to this folder:

1. **ocean.mp3** - Ocean waves loop
2. **rain.mp3** - Gentle rain loop
3. **forest.mp3** - Forest birds and nature ambience
4. **fire.mp3** - Crackling fireplace loop
5. **stream.mp3** - Brook/stream flowing water
6. **cricket.mp3** - Night crickets loop
7. **wind.mp3** - Gentle wind through trees
8. **cafe.mp3** - Coffee shop ambience
9. **thunder.mp3** - Thunderstorm with rain

## Where to Download:

### Recommended: Freesound.org
- https://freesound.org
- Search for "[sound name] loop" (e.g., "ocean waves loop")
- Download as MP3 format
- Look for CC0 (public domain) or CC-BY licenses

### Alternative: Pixabay
- https://pixabay.com/sound-effects/
- Free for commercial use
- No attribution required

## File Requirements:

- **Format**: MP3
- **Duration**: 30 seconds to 2 minutes (will loop automatically)
- **Quality**: At least 128kbps
- **Size**: Keep under 5MB per file for fast loading

## Tips:

- Look for "seamless loop" or "loopable" in the description
- Test the loop point to ensure it sounds continuous
- Prefer mono or stereo files (avoid surround sound)
- Normalize volume levels so all sounds are similar loudness
